! Due to uBlock Origin limitations, the rules below can only block URLs with the specific file extension only containing HTML content.
! File extensions: cnf, conf, cfg, log, asc, txt, bak, dat, dsk, raw
/^http://.+(/.+)?/.+\.(?:(?:[cC][oO]?[nN][fF])|(?:[cC][fF][gG])|(?:[lL][oO][gG])|(?:[aA][sS][cC])|(?:[tT][xX][tT])|(?:[bB][iI][nN])|(?:[dD][aA][tT])|(?:[dD][sS][kK])|(?:[rR][aA][wW]))([#?].*)?$/$all
