! Due to uBlock Origin limitations, the rules below can only block URLs with the specific file extension only containing HTML content.
! File extensions: exe, apk, app, appx, appxbundle, elf, bin, com, bat, gadget, msi
/^http:\/\/.+(/.+){0,}/.+\.(?:(?:[eE][xX][eE])|(?:[aA][pP][kK])|(?:[aA][pP][pP][xX]?([bB][uU][nN][dD][lL][eE])?)|(?:[bB][iI][nN])|(?:[cC][oO][mM])|(?:[bB][aA][tT])|(?:[gG][aA][dD][gG][eE][tT])|(?:[mM][sS][iI]))([/?].*)?$/$all
