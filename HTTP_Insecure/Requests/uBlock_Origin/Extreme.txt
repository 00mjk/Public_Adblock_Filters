! Guidance: Some of these lists are an usable version of uBO hard mode and/or medium mode. See https://github.com/gorhill/uBlock/wiki/Blocking-mode.
! Blocks all HTTP websites from untrusted TLDs.
http://*$all,domain=~gov|~int|~edu|~mil|~org|~us|~com|~net|~io|~uk|~ca|~nz|~au|~mx|~info|~eu|
! This will cause conflicts with if using uBlock Origin, Policy Control or uMatrix with Adguard to filter out Javascript.
! Block Javascript on first-party domains on HTTP websites from trusted TLDs.
http://*$~third-party,script,domain=gov|int|edu|mil|org|us|com|net|io|uk|ca|nz|au|mx|info|eu|
! Allows only third-party passive content to load on HTTP websites from trusted TLDs.
http://*$third-party,~stylesheet,~image,~media,domain=gov|int|edu|mil|org|us|com|net|io|uk|ca|nz|au|mx|info|eu|
! Block Javascript on untrusted TLDs on first-party domains for HTTPS websites. 
https://*$~3p,script,domain=~gov|~int|~edu|~mil|~org|~us|~com|~net|~io|~uk|~ca|~nz|~au|~mx|~info|~eu|
! Block third-party Javascript and frames on untrusted TLDs on third-party domains for HTTPS websites. More useable version of uBO medium mode for HTTPS.
https://*$3p,script,frame,domain=~gov|~int|~edu|~mil|~org|~us|~com|~net|~io|~uk|~ca|~nz|~au|~mx|~info|~eu|
