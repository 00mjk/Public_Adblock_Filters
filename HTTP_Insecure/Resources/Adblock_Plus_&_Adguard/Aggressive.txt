! Guidance: Some of these lists are an usable version of uBO hard mode and/or medium mode. See https://github.com/gorhill/uBlock/wiki/Blocking-mode
! Blocks Javascript and frames on all HTTP websites forces first-party domains.
! This will cause conflicts with if using uBlock Origin, Policy Control or uMatrix with Adguard to filter out Javascript.
http://*$~third-party,script,subdocument
! Allows only third-party passive content to load on HTTP websites.
http://*$third-party,~stylesheet,~image
!  Blocks Javascript and frames from untrusted TLDs on HTTPS websites from first-party domains.
https://*$script,subdocument,domain=~gov|~int|~edu|~mil|~org|~us|~com|~net|~io|~uk|~ca|~nz|~au|~mx|~info|~eu
