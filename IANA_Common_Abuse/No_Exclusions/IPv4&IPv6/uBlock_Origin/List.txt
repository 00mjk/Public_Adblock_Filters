! Description: Intended for devices that support IPv4 and IPv6 traffic with uBlock Origin. List of IP addresses that are commonly abused from LAN or private network according to IANA that should be blocked for to increase security against network attacks from outbound traffic.
! Use them at your own risk. 
! Attribution and Sources: gwarser (https://github.com/gwarser/filter-lists/blob/master/lan-block.txt)
! ========================================================================================
! Blocking commonly abusIP addresses in uBlock Origin or NanoDefender
! Outbound requests to these IP addresses should be blocked
! unless for a specific reason such as mangaging network administration, an external server, etc.
! Guidance: IANA Abuse (https://www.iana.org/abuse) and (https://www.iana.org/help/abuse-answers)
! Multicast, reserved and other unallocated IP addresses are excluded since they are used for legitimate purpose, see https://www.iana.org/assignments/ipv4-address-space/ipv4-address-space.xml.
! ==============================================================================================================================================
! === Stateless Address AutoConfiguration or Link-Local Addresses ==== 
! IP ranges is intended when your device is not set to static address or DHCP server. 169.254.0.0/16 and fe80::/10 ranges.
/^([a-zA-Z]{2,20})://(?:169\.254\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])))?:?(?:6553[0-5]|655[0-2][0-9]|65[0-4][0-9]{2}|6[0-4][0-9]{3}|[1-5][0-9]{4}|[1-9][0-9]{1,3}|[1-9])?)[:/]$/$all
! === Private IPv4 Address Range ======
! These ranges are used for IP addresses associated for a private network. 
/^([a-zA-Z]{2,20})://(?:10\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9]))?:?(?:6553[0-5]|655[0-2][0-9]|65[0-4][0-9]{2}|6[0-4][0-9]{3}|[1-5][0-9]{4}|[1-9][0-9]{1,3}|[1-9])?)[:/]$/$all
/^([a-zA-Z]{2,20})://(?:172\.(?:3[01]|2[0-9]|1[6-9])\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9]))?:?(?:6553[0-5]|655[0-2][0-9]|65[0-4][0-9]{2}|6[0-4][0-9]{3}|[1-5][0-9]{4}|[1-9][0-9]{1,3}|[1-9])?)[:/]$/$all
/^([a-zA-Z]{2,20})://(?:192\.168\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9]))?:?(?:6553[0-5]|655[0-2][0-9]|65[0-4][0-9]{2}|6[0-4][0-9]{3}|[1-5][0-9]{4}|[1-9][0-9]{1,3}|[1-9])?)[:/]*$/$all
! === Localhost Ranges =======
! Websites like leoaffairs.com use localhost to bypass your adblocker. So, this is needed to block loopholes in regards to adblocking.
|^([a-zA-Z]{2,20}):(?:127\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])\.(?:25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9]))?:?(?:6553[0-5]|655[0-2][0-9]|65[0-4][0-9]{2}|6[0-4][0-9]{3}|[1-5][0-9]{4}|[1-9][0-9]{1,3}|[1-9])?)[:/]/$all
