! Description: Intended for devices that support IPv4 and IPv6 traffic for other Adguard software. List of IP addresses classified by IANA that are associated with the private network or LAN that should be blocked for to increase security against network attacks from outbound traffic. Localhost IP addresses are exempt. Exclusions are added to this list.
! Use them at your own risk.
! Attribution: gwarser
! =========================================================================================
! Blocking commonly abused reserved IP addresses in Adguard
! Outbound requests to these IP addresses should be blocked
! unless for a specific reason such as mangaging network administration, an external server, etc.
! Guidance: IANA Abuse - https://www.iana.org/abuse and https://www.iana.org/help/abuse-answers)
! Multicast and other unallocated (reserved) IP addresses are excluded since they are used for legitimate purposes or not common, see https://www.iana.org/assignments/ipv4-address-space/ipv4-address-space.xml
! ==============================================================================================================================================
! This list does not block traffic for the following protocols: ICMP, ICMPv6, DHCP, DHCPv6, ARP, etc.
! === Stateless Address AutoConfiguration or Link-Local Addresses ==== 
! 169.254.* and [fe**:*] IP ranges is intended when your device is not set to static address or DHCP server. 
169.254.*^$network
! For full protection from [fc**:*] and [fd**:*] addresses please check IPv6Raw-List.txt.
! === Private IPv4 & IPv6 Address Range ======
! These ranges are used for IP addresses associated for a private network. 
10.*^$network
172.16.*^$network
172.17.*^$network
172.18.*^$network
172.19.*^$network
172.2*.*^$network
172.30.*^$network
172.31.*^$network
192.168.*^$network 
! ==== Common Exclusions ======
! These are examples of IP addresses used for legitimate purposes
! Localhost exclusions are excluded.
! Login to router or gateway.
@@10.0.0.1:80^$network
@@10.0.0.1:443^$network
@@10.0.1.1:80^$network
@@10.0.1.1:443^$network
@@172.16.0.1:80^$network
@@172.16.0.1:443^$network
@@172.16.1.1:80^$network
@@172.16.1.1:443^$network
@@192.168.0.1:80^$network
@@192.168.0.1:443^$network
@@192.168.1.1:80^$network
@@192.168.1.1:443^$network
